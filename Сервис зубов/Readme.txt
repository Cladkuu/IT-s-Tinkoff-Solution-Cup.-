
Описание полей в ERD:
  Таблица Зубы
- id : id зуба
- Статус наличия зуба : Есть ли зуб
- Статус здоровья зуба : Здоров или нет
- Size : размер
- цвет
- Тип зуба: человечески, искусственный
- Вид зуба
- id зубной полости
- URL фото зуба в файловом хранилище

  Таблица Зубная полость:
  - id : id зубной полости
  - status
  - id ребенка : Ребенок, которому принадлежит зубная полость
